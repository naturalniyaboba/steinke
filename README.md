# Steinke Immobilien

Официальный сайт компании Steinke Immobilien - профессионального агентства недвижимости в Лейпциге.

## О проекте

Современный веб-сайт для агентства недвижимости, разработанный с использованием современных технологий веб-разработки.

## Технологии

- **Vite** - быстрый инструмент сборки
- **React** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Tailwind CSS** - утилитарный CSS фреймворк
- **shadcn/ui** - компоненты пользовательского интерфейса

## Разработка

### Требования

- Node.js 20 или выше
- npm

### Установка

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу `http://localhost:8080`

### Сборка для production

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Предпросмотр production сборки

```bash
npm run preview
```

## Деплой

Проект автоматически деплоится на GitHub Pages при пуше в ветку `main` через GitHub Actions.
